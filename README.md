Utility Addons:
===========
 * In-Game Settings Buddy
  * It Enables/Disables "Game Settings" from the Loader
    * Enable Anti AFK (F1).
    * Enable Extended Zoom (F2)
    * Slider to set the Zoom Value.
    * Enable Movement Hack (F3). [DISABLED]
    * Enable Tower Ranges Draws (F9).
    * Enable InGame Chat (F7).
    * Draw Watermark (F4).

 * Item Swapper Buddy
  * It swaps Items using the Numpad
    * 4 - Item Slot 1
    * 5 - Item Slot 2
    * 6 - Item Slot 3
    * 1 - Item Slot 4
    * 2 - Item Slot 5
    * 3 - Item Slot 6
    * 0 - Reset
